# 🎓 Edu Shop

A full-stack educational e-commerce platform built for learning modern web development technologies.

---

## 🚀 Tech Stack

### Frontend
- ⚛️ React
- ▲ Next.js
- TypeScript
- Tailwind CSS
- Axios

### Backend
- 🐍 Django
- Django REST Framework
- JWT Authentication
- SQLite (Development)
- PostgreSQL (Production Ready)

### DevOps
- Docker
- Docker Compose
- Git & GitHub

---

## ✨ Features

- User Authentication
- JWT Login/Register
- Course Management
- Shopping Cart
- Order System
- User Profile
- Admin Dashboard
- REST API
- Responsive UI

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/aghaderi78/edu-shop-new.git
cd edu-shop-new
```

---

## Backend Setup

```bash
cd edu-shop-backend/backend

python -m venv venv
source venv/bin/activate      # Linux/macOS
venv\Scripts\activate         # Windows

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

---

## Frontend Setup

```bash
cd edu-shop-frontend
npm install
npm run dev
```

---

## Docker

```bash
docker compose up --build
```

---

## Environment Variables

Create a `.env` file based on `.env.example`

---

## API

Backend: http://localhost:8000  
Frontend: http://localhost:3000
